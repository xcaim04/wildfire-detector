# Como ejecutar el proyecto

```bash
git clone https://github.com/xcaim04/wildfire-detector.git
cd wildfire-detector
pip install -r requirements.txt
```

El codigo va en la carpeta src/

nada de lo demas se toca.

Si quieren levantar le server, ejecutan `uvicorn src.main:app --reload` desde la raiz del proyecto en la terminal
